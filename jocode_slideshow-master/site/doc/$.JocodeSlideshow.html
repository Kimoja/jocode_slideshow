    • $loader <String>

      ...

    • $pause_hover <String>

      The selector, relative to the initialization container, which defines the elements to listen to trigger the break, on the "hover" event. If "<" is specified, then the initialization container will be used.

    • $mask_buttons_hover <String>

      The selector, relative to the initialization container, that defines the elements that triggers whether to display buttons, on the "hover" event. The container of buttons must be different of the container of the slideshow. If "<" is specified, then the initialization container will be used.

    • $mask_nav_hover <String>

      The selector, relative to the initialization container, that defines the elements that triggers whether to display the navigation container, on the "hover" event. The parameter container of the navigation must be defined and must be different of the container of the slideshow. If "<" is specified, then the initialization container will be used.

    • mask_buttons_load <Boolean>

      Hide buttons during loading of a slide. The container of buttons must be different of the container of the slideshow.

    • mask_buttons_fx <Boolean>

      Hide buttons during transition of a slide. The container of buttons must be different of the container of the slideshow.

    • mask_nav_load <Boolean>

      Hide navigation container during transition of a slide. The parameter container of the navigation must be defined and must be different of the container of the slideshow.

    • mask_nav_fx <Boolean>

      Hide navigation container during transition of a slide. The parameter container of the navigation must be defined and must be different of the container of the slideshow.
